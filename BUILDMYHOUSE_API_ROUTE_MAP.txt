# BuildMyHouse API Route Map

Last updated: 2026-06-12
Global prefix: /api (set in apps/backend/src/main.ts)
Base URL (production): https://api.buildmyhouse.app/api

Guard legend:
- Public — no auth required
- JWT — JwtAuthGuard (Bearer token)
- Roles — JwtAuthGuard + RolesGuard + @Roles(...)

## Important notes

1. Two contractor APIs exist:
   - /api/contractors — project/GC onboarding, requests, Stripe Connect, verification
   - /api/marketplace/contractors — marketplace directory profiles

2. WebSocket (Socket.IO) handles realtime chat/notifications — not REST routes above.

3. Rate limiting (@Throttle) on auth routes; webhooks skip throttle.

4. Production runs prisma migrate deploy on container start before boot.
